This language pack contains Bulgarian translations for Flarum (compatible with `2.0.0` or newer) and popular extensions. Full list of supported extensions is available below.


## Installation

You can install language pack using [Composer](https://getcomposer.org/):

```console
composer require "flarum-lang/bulgarian:@dev"
```

Then enable extension in admin panel of your Flarum.


## Updating

You can update language pack using [Composer](https://getcomposer.org/):

```console
composer update flarum-lang/bulgarian
```

Then clear the cache:

```console
php flarum cache:clear
```
